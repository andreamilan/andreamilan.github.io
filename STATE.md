# Current State

- The project is in a stable live state.
- The site is a dark, single-page editorial personal website for Andrea Milan.
- Deployment to GitHub Pages is working through GitHub Actions.
- The current product direction is stable enough to pause without structural follow-up work.

## Confirmed Direction
- Personal editorial feeling over SaaS-like portfolio structure
- Strong hero with portrait and concise first-person positioning
- Overview as the bridge into the career narrative
- One featured current role, with previous roles kept compact
- Education separated from Experience
- Principles used as the judgment/leadership framing section
- Get in touch kept light, with LinkedIn primary and Medium secondary
- No email, no contact form, no standalone About/Writing/Contact pages

## Recent Refinements Reflected in the Live Site
- Header/top bar simplified and made quieter
- Hero spacing refined, especially on mobile
- Multilingual greeting reintroduced in a subtle rotating label below the portrait
- Experience compact rows improved for spacing and scanability
- Education made lighter and more secondary
- Principles reframed and polished as a stronger authored section
- Shared section frame and text measure made more consistent
- Mobile metadata layout refined so date and location read clearly
- Portrait performance and typography consistency improved
- Codebase cleaned up to remove obsolete folders and unused files
